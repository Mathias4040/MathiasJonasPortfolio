About the Project

About SiteSpy

SiteSpy is a mobile-first construction project records and cost estimation app built with Expo React Native, Firebase, and Firestore. It helps construction teams capture measurements, estimate material costs, and review project history on any device — even in low-connectivity field conditions.

React Native Expo Firebase Firestore
My Contribution

My Contribution

My work covers how SiteSpy users return to and review their past projects. I built ProjectHistoryScreen.js to list all saved project sessions in chronological order with clear metadata, ProjectDetailsScreen.js to display the full project record when a user taps into a history entry, and worked with projectService.js to ensure efficient data fetching.
