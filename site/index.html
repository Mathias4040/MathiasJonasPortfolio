About the Project

About SiteSpy

SiteSpy is a mobile-first construction project records and cost estimation app built with Expo React Native, Firebase, and Firestore. It helps construction teams capture measurements, estimate material costs, organise site records, and review project history on any device — even in low-connectivity field conditions.

My Contribution

My Contribution

My work covers how SiteSpy users return to and review their past projects. I built ProjectHistoryScreen.js to list all saved project sessions in chronological order with clear metadata, ProjectDetailsScreen.js to display the full project record when a user taps into a history entry, and worked with projectService.js to ensure the data fetching behind these screens is efficient and handles empty states gracefully.

Reflection

My Reflection

Project history is the feature that makes SiteSpy valuable beyond a single use — users return to compare estimates from different dates. I learned how to paginate Firestore queries so history lists do not slow down as the dataset grows. Making detail views readable on small screens required careful typographic hierarchy choices.
